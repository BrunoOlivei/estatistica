## Moda

$$

> [!Note]
> Se o conjunto de dados apresentar todos os elementos com a mesma frequ√™ncia absoluta, n√£o existir√° moda. Se ocorrer v√°rias frequ√™ncias iguais, ent√£o teremos uma distribui√ß√£o com mais de uma moda, portanto o calculo deve ser realizado para cada classe que apresente o maior n√∫mero de frequ√™ncia absoluta.
